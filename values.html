Values

Joe Legner

September 19, 2023

This is a pretty good list of things I value. When I use the word value here, I mean desired quality of ongoing action. Values are not goals. They involve continuous behavior. Being loved is not a value. It is a goal. You can check it off your list when it happens. Kindness is a value. You can never check it off the list. You can behave with kindness. Kindness is a quality of action, so in my system it can be called a value. Here are the rest:
