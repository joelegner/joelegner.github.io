It is easy to mistake values for goals. Values are not the same thing as goals. They involve continuous behavior. A goal can be checked off. No value can be checked off. A goal can be achieved. No value cannot be achieved. A goal is a concrete result. A value is an abstract quality of something else ‚Äì my behavior.

And what is behavior? Behavior includes anything that I do that can be seen or heard doing. Behavior is overt. Behavior is public. If I was being filmed, the film camera and microphones would capture my behavior. It would not capture my thoughts, even if I said them out loud. Saying them is behavior. Thoughts are covert. Thoughts are private.

Let‚Äôs see a contrast. Being loved is not a value. It is a goal. You can check it off your list when it happens. On the other hand, kindness is a value. You can never check it off the list. You can behave with kindness.

Kindness, on the other hand, is a quality of action. So in my system it can be called a value.
